# Setup Instructions

## Prerequisites
- Node.js (for frontend)
- Python 3.8+ (for backend)
- Git

## Getting Started

1. Clone the repository:
   ```
   git clone https://github.com/akshitjmsb/usacrickethub.git
   cd usacrickethub
   ```

2. See `frontend/` and `backend/` folders for specific setup instructions.
