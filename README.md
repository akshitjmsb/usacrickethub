# USA Cricket Hub

Welcome to USA Cricket Hub – the go-to platform for cricket lovers in the USA!

## Vision
To connect, inform, and empower the US cricket community with news, scores, resources, and a vibrant online community.

## Project Structure
- `backend/` – FastAPI backend (APIs, user management, etc.)
- `frontend/` – Next.js frontend (website, SSR, SEO)
- `docs/` – Documentation and planning

## Getting Started

1. Clone this repo
2. See `docs/` for setup instructions
